# Replace Humanity Project

Welcome to the Replace Humanity documentation — the next generation of `Automation`.

## Introduction
ReplaceHumanity Team has developed a next-generation task automation system that does not rely on traditional methods such as RPA, which are typically limited to automating websites. With our system, you can automate your entire computer and work with any application of your choice, enabling true multitasking and comprehensive automation across programs.

### What is Replace Humanity Project?
<span style={{ fontWeight: 'bold', fontFamily: "'Open Sans', sans-serif" }} >Replace Humanity Project</span> is the foundation of a new era of task automation software. It allows you to automate any task on a computer with ease and is adaptable to other computers, as long as the screen resolution size remains the same, since it uses OCR (Optical Character Recognition) to detect words and Image Recognition to locate icons on screen. With Replace Humanity Project, anyone can automate computer tasks and share the automation file with others so they can run the same tasks on their systems. It can also be assigned to any LLM to provide it with practical skills, such as ChatGPT or Google Gemini. It acts as a “body” for LLMs like ChatGPT and Google Gemini, enabling them to perform real actions on a computer rather than only generating text.

### What is a Specific Task Performer (STP)?
Specific Task Performer (STP) is a task automation framework designed to build complete, scenario-based learning models that remember which step or action to perform in a specific situation. This enables fully automated workflows without human interaction and aims to ensure a zero-error rate. During training, the model is taught how to handle different scenarios and anomalies effectively. You can use Assets to access reusable code components and speed up development. You can also visit the Marketplace to purchase ready-to-use STPs designed by Skill Creators.
<br /><br />Whether you are an individual developer or part of a larger team, Replace Humanity helps you build autonomous, dynamic, and high-speed, production-grade workflow and task automation programs. You simply need to turn on the computer and send an activation signal—such as a message in Teams—to start a specific task with an Excel file, data entry process, data scraping job, or other workflow. The bot will then execute and complete the task automatically.
<br />Upon completion, it sends you a message prompting you to proceed with another task or shut down the computer for the day.
